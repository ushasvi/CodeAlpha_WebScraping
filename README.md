# CodeAlpha Web Scraping Project

## Project Overview
This project demonstrates web scraping using Python. Book information such as title, price, and rating is extracted from the Books to Scrape website and stored in a CSV file.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Extracts book titles
- Extracts prices
- Extracts ratings
- Saves data into CSV format

## How to Run
1. Install required libraries:
   pip install requests beautifulsoup4 pandas

2. Run:
   python books_scraper.py

3. Output file:
   books_data.csv
